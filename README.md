# ✊✋✌️ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors Game** built using **HTML, CSS, and JavaScript**.  
Players can compete against the computer and instantly see the result.

## 🎮 Game Rules

- **Rock beats Scissors**
- **Scissors beats Paper**
- **Paper beats Rock**

The player selects an option and the computer randomly selects one. The result is displayed immediately.

## ✨ Features

✔ Play against computer  
✔ Random computer choice
